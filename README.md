# New Hira 985 Combine Harvester Website

Production-ready static website for **New Hira Crop Challengers, Nabha**.

## Included

- Responsive landing page for New Hira 985 combine harvester
- English, Hindi and Punjabi language switcher
- Your uploaded posters integrated into the gallery
- Custom SVG campaign poster
- Product feature sections, price/options calculator and FAQ
- Searchable spare parts catalogue
- Lead enquiry form with Firebase Firestore support and local fallback
- WhatsApp, call and email CTAs
- PWA files: manifest and service worker
- SEO meta tags, Open Graph image, JSON-LD business schema
- Print-friendly brochure styling

## Main files

- `index.html` — page structure and SEO
- `styles.css` — production responsive design
- `script.js` — language switching, quote calculator, parts search, gallery and form
- `firebase-config.js` — Firebase web config
- `firestore.rules` — write-only lead collection rules

Upload the folder contents to GitHub Pages, Firebase Hosting, Netlify, Vercel, cPanel, or any static host.

- Open Graph URL if you add one
